## 🌟 Vision
**MediScan AI** is a decentralized healthcare infrastructure designed to bridge the gap between advanced neural-linked computer vision and clinical diagnostic accuracy. By leveraging the power of Google's Gemini Pro and high-performance edge computing, MediScan AI provides real-time, encrypted medical data synthesis for the modern world.

## 🚀 Key Features

### 🔍 Neural Vision Scanning
- **OCR Integration**: Utilizing `Tesseract.js` for high-fidelity clinical document parsing.
- **Image Synthesis**: Real-time analysis of medical imagery using computer vision clusters.
- **Confidence Scoring**: Dynamic probability mapping for diagnostic accuracy.

### 🧠 ARIA: AI Research & Intelligence Assistant
- **Gemini Pro Integration**: Powered by Google's state-of-the-art LLM for medical data synthesis.
- **Contextual Awareness**: Intelligent chat interface for complex patient history analysis.
- **Encrypted Data Link**: Secure AES-256 data orchestration for patient privacy.

### 📊 Comprehensive Health Dashboard
- **Live Telemetry**: Real-time monitoring of system core status and network latency.
- **Patient History Clusters**: Segmented data visualization for long-term health tracking.
- **Lab Result Analytics**: High-performance charting using `Chart.js` and `Recharts`.

### 📍 Clinic Finder & Network
- **Interactive Mapping**: Powered by `Leaflet` and `React-Leaflet` for regional clinic localization.
- **Infrastructure Status**: Real-time availability of decentralized medical nodes.

## 📐 Architecture
MediScan AI operates on a **Decentralized Edge Cluster** model:
1. **Data Ingestion**: Clinical data is captured via vision modules or OCR.
2. **Neural Processing**: Gemini Pro synthesizes raw data into actionable insights.
3. **Encrypted Storage**: Results are stored in isolated Firestore clusters with biometric-linked security rules.
4. **Interface Layer**: A high-tech, responsive UI (Navy-themed with Medical Teal accents) provides the diagnostic cockpit.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v20+)
- Gemini API Key

### Local Installation
1. **Clone the Repository**
   ```bash
   git clone https://github.com/tejaskumarg06-ctrl/mediscan-AI.git
   ```
2. **Install Dependencies**
   ```bash
   npm install --force
   ```
3. **Environment Configuration**
   Create a `.env.local` file:
   ```env
   GEMINI_API_KEY=your_key_here
   ```
4. **Launch Development Server**
   ```bash
   npm run dev
   ```
